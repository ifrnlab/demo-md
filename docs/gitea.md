## Instalação

A instalação do Gitea foi feita usando o comando `docker compose`. Seguimos os seguintes passos:

1. `git clone https://github.com/docker/awesome-compose`
2. `cd awesome-compose/gitea-postgres`
3. `docker compose up -d`
